# Django Bus Booking API

A production-ready bus booking backend built with Django REST Framework.

## Features

- JWT Authentication
- Bus Management
- Seat Management
- Seat Booking with transaction safety
- Dynamic available seat calculation

## Tech Stack

- Python
- Django
- Django REST Framework
- JWT
- SQLite (Dev)

## Run Locally

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
